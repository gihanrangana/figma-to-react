# figma-to-react
This repo includes how to import figma template directly to react project using figma api

# Demo

[Demo](https://figma-to-react-zeta.vercel.app/)

## Install
```
npm install 
npm run dev
```

```
yarn && yarn dev
```

## Usage

```ts
const figma = useFigma()

return (<FigmaToReact figma={figma}/>)
```
