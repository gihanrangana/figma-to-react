# figma-to-react
This is frontend tool to conver figma designs to react jsx elements using figma api

# Demo

[Demo](https://figma-to-react-zeta.vercel.app/)

## Install
```
npm install 
npm run dev
```

```
yarn && yarn dev
```

## Usage

```ts
const figma = useFigma()

return (<FigmaToReact figma={figma}/>)
```
